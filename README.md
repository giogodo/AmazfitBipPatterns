# AmazfitBipPatterns
Color patterns for gimp composed of a palette of 8 colors compatible with the Amazfit Bip smartwatch.

![Amazfit Bip 4 bit palette](Palette.PNG?raw=true "Amazfit Bip 4 bit palette")  
Amazfit Bip 4 bit palette.

The new palette created with the patterns allows to render up to 55 different colors.  

![Example of implementation](Example.gif?raw=true "Example of implementation")  
  [Example of implementation](https://amazfitwatchfaces.com/bip/view/?id=19009).

The repo includes a [test watchface](TestWatchface/colortest_packed.bin), with every posible single color.
